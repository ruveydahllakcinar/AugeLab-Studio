AUGELAB İLE FİND OBJECT VE OBJECT COUNTER (Nesne Bulma ve Sayma)

Günümüz Endüstrisinde ve fabrikalarda artık teknoloji yerini almış durumda. Bir pet şişe fabrikası düşünelim günlük ürettiği şişe sayısını saymak istiyoruz. Bunu teknolojiyi Görüntü işlemeyi kullanarak yapabiliriz. Tabii kod yazmamız gerekecektir. Daha kolay yolu ise bu sistemi AUGELAB Studio yoluyla oluşturmak olabilir. Bugün ki yazımda AUGELAB ile Nesne bulma ve bulunan nesneyi saymayı deneyeceğiz.

FİND OBJECT

Öncellikle Augelab Studio’yu açalım.

Nesne bulma ve sayabilmek için Studio’nun içine aktif işleyen video bloğunu eklememiz gerekecek.

Gerçek sistemde kamera bloğu kullanıyoruz ama şu anda bir video üzerinden işlem yapacağımız için Camera USB/Video bloğunu ekleyerek open video file kısmından videomuzu seçiyoruz.

Üzerinde çalıştığınız videolar yüksek çözünürlükte olabilir. Bu durum bilgisayarınızın performansını etkileyebilir. Düşük çözünürlükte çalışmak sizin yazılımınızı etkilemiyorsa image resizer kullanabilirsiniz. Bu sebepten dolayı videonun çözünürlülüğünü sisteme çok eksisi olmadan bilgisayarınızı rahatlatmak amacıyla çözünürlük kontrolü sağladığımız image resizer eklememiz gerekecek. Output image çıkışına image resizer bloğunun girişine bağlayıp içindeki değeri 25 yapıyoruz.

Resized image çıkışına durumu görmek için Show image ekleyip f5 e basarak sistemi bir kere çalıştırıyoruz.

Nesne bulmak amacıyla ilk başta hangi nesneyi bulmamız gerektiğini öğretmemiz lazım. Bunun için image memory ile fotoğrafı tutup sistemi o fotoğraf ile besleyebiliriz. İmage resizer ile işlediğimiz veriyi bu bloğun çıkışıyla image memory bloğunun input img girişine bağlıyoruz. Hold condition girişi sayesinde görüntüyü tutacağız. Mavi input için logic input bloğunu ekleyip bağlıyoruz. Ve image memory bloğunun çıkışına görüntü seçmek için Image ROI select bloğunu ekleyip bağlıyoruz.

Logic input bloğunu True yaptıktan sonra sistemi çalıştırıp uygun olan fotoğrafa geldiğinde logic inputu false seçeneğine çekerek resmi sabitliyoruz. (Eğer sistem burada hata verirse image select bloğunun sonuna Show image ekleyerek sorunu giderebilirsiniz.)

Image resizer çıkışına bir tane find object bloğu ekleyeceğiz. Find Object çalışma mantığı bir tane main image alıyor, bu görüntü kameradan anlık da gelebilir ya da bir video dosyası da olabilir. Bir tane de object image girişi vardır. Bu girişe gelen veri, referans görüntümüz veya seçmiş olduğumuz görüntü olabilir. ROI select bloğu ile seçtiğimiz görüntüyü de find object bloğuna aktarmamız gerekiyor. Bu yüzden find object bloğunun object image girişini ROI image çıkışı ile bağlıyoruz. Son olarak elde ettiğimiz çıktıyı görmek için Show image ekliyoruz.

Find object bloğunda bulunan match Threshold ile modele en uygun doğruluk oranını yakalamak için bulunan bar’ı deneyerek modele en uygun seviyede sabit bırakıyoruz.
